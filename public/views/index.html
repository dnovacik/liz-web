Welcome to LiZ

LIZ is a cryptocurrency and a secure decentralized platform designed as a general purpose tool allowing better and secure way to conduct businesses.

LIZ is built upon PIVX with several additions and modifications. LIZ consensus is achieved through a POS 3-tier network. An onchain layer inspired by dBFT type consensus allows users to vote for existing addresses and share reputation.

A LIZ identity is a LIZ address and a democratically voted score.

LIZ is giving itself to the communities and offer project managers, freelancers and authors the possibility to open their histories and gain reputation while giving investors an easy way to evaluate a project as legitimate. LIZ comes with an online growing database connected to its blockchain. Each profile on the LIZ database is linked to a unique LIZ address, email address and a Bitcointalk account.

This is just the begining. LIZ is a tool. Use it. Improve it.

LIZ is using masternodes in orded to secure its network. Masternodes are performing several operations such as serving the chain to other nodes, allowing instant & anonymous transactions. The masternode collateral is 10000 LIZ.
